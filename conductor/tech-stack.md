# Technology Stack

## Overview
This project leverages a modern web development stack to deliver a high-performance and user-friendly portfolio website.

## Core Technologies

### Programming Language
- **TypeScript**: A superset of JavaScript that adds static typing, enhancing code quality and maintainability.

### Frontend Frameworks
- **Next.js**: A React framework for building full-stack web applications. For this project, it will be configured for **Static Site Generation (SSG)** to produce a fully static website.
- **React**: A declarative, component-based JavaScript library for building user interfaces.

### Styling and UI Libraries
- **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.
- **Radix UI**: A collection of unstyled, accessible UI components for building high-quality design systems.
- **Shadcn UI**: A collection of reusable components built with Radix UI and Tailwind CSS, providing a beautiful and customizable UI foundation.

## Deployment
- **GitHub Pages**: The portfolio will be deployed as a static website on GitHub Pages. This is achieved by using Next.js's Static Site Generation capabilities.
